# MisProfesiones
Addon minimalista y ultra-estable desarrollado para **World of Warcraft: Wrath of the Lich King (v3.3.5a)**. Está diseñado específicamente para solventar las limitaciones del cliente clásico de WoW en servidores privados modificados que permiten disponer de múltiples o todas las profesiones simultáneamente (**All-Prof / Todas las Profesiones**).
